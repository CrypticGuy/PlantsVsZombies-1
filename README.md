# Plants VS Zombies

## How to start?
1. Clone the repo on your desktop.<br>
2. You must have JavaFX installed and the .jar files must be present in `/usr/share/openjfx/lib` for the game to run.<br>
3. Move to `src`.<br>
4. Run `make` in src and then `make run`.<br>
5. Enjoy.<br>
